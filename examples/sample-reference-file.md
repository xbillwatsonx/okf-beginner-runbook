# Seed Tray Size Reference

Use this reference when choosing a tray size for a small microgreens batch.

## Common Sizes

- 10x20 tray - Standard production tray for many microgreens.
- 5x5 tray - Useful for tests, samples, or tiny batches.
- 2-inch nursery pot - Useful for small experiments.
