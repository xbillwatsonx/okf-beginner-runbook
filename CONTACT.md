# Contact

For questions, suggestions, issues, or improvements, use the GitHub repository for this project:

https://github.com/xbillwatsonx/okf-beginner-runbook

The best ways to help are:

- Open a GitHub issue for bugs, unclear instructions, or suggested improvements.
- Open a pull request if you have a specific fix or addition.
- Share examples of how you adapted the runbook so the project can improve over time.

Please do not include private credentials, private files, API keys, or sensitive personal information in public GitHub issues or pull requests.
