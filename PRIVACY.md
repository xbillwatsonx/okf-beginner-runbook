# Privacy

This repository is a static open-source project. The files in this package do not collect personal information, run a hosted service, create user accounts, set cookies, or send data anywhere.

If you download or clone this repository, any files you create with the runbook stay wherever you put them unless you choose to share, publish, upload, or sync them with another service.

## GitHub

This project is hosted on GitHub. If you visit, star, fork, clone, open an issue, or submit a pull request, GitHub may process information according to its own terms and privacy policies.

## Your Knowledge Base

The starter kit and prompts may be used with AI agents or tools of your choice. Those tools may have their own privacy behavior. Before using private notes, customer information, credentials, business records, or other sensitive material with an AI tool, check that tool's privacy settings and data handling rules.

## Public Contributions

GitHub issues and pull requests are public by default. Do not post API keys, passwords, private documents, customer data, or sensitive personal information.
